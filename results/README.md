# Results Directory

Experiment outputs organized by study phase:

```
results/
├── prelim/          # Preliminary study
│   ├── metrics/     # CSV: QWK, Pearson, RMSE, MAE
│   └── predictions/ # OOF predictions per experiment
└── main/            # Main research
    ├── metrics/
    ├── predictions/
    └── figures/      # Generated plots for paper
```

Regenerate by running notebooks in `notebooks/`.
